# Frontend (SvelteKit) Requirements

## Rules
1. Use SvelteKit with Server-Side Rendering (SSR).
2. Style with Tailwind CSS.
3. Load the project name (`PROJECT_NAME`) dynamically from environment variables to display in the UI/Navbar.
4. Use standard `fetch` API for communicating with the Go backend, utilizing `VITE_API_URL` from the `.env` file.
